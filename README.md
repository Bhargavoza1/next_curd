## Basic crud with next.js 13 and mongoDB


## Setup

I am using local mongodb with compass
and postman to test api
make .env or .env.local file and paste 
```text
    MONGODB_URI="mongodb://127.0.0.1:27017/crud_db"
```

 ## getting start

```text
    npm run dev
```
